# Simple .Net Core Template with EFCore and SQLite
This is a basic .NET Core web application template, ideal for smaller projects.

##Technology Stack
.NET Core 8.0
SQLite
EF Core 8.0

##Features Included
Pre-implemented Repository and Unit of Work patterns with example usage
Integrated logging using Serilog
Sample seed data for quick setup
Standard .NET Core .gitignore file included
